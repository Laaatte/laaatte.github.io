About

This Jekyll blog was created just the way I wanted. It's not perfect, but I hope you enjoy it. Also, GPT was a helpful friend along the way.

What I Use

  • Jekyll for static site generation
  • Hosted on GitHub Pages
  • JavaScript & jQuery for dynamic pagination and search functionality
  • Custom Pagination for efficient navigation through posts (Implemented with jQuery)
  • Ruby script to generate posts.json instead of using pagination gem
  • Liquid templating language for dynamic content generation

사용한 것들

  • 정적 사이트 생성을 위한 Jekyll
  • GitHub Pages에서 호스팅
  • 동적 페이지네이션 및 검색 기능을 위한 JavaScript & jQuery
  • 효율적인 포스트 네비게이션을 위한 커스텀 페이지네이션 (jQuery로 구현됨)
  • 페이지네이션 젬 대신 posts.json을 생성하는 Ruby 스크립트
  • 동적 콘텐츠 생성을 위한 Liquid 템플릿 언어
